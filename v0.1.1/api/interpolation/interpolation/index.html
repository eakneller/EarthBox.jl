Interpolation

Initialization

EarthBox.Interpolation.initialize! — Function
initialize!(model::ModelData; kwargs...)::Nothing

Initialize interpolation option parameter flags.

Arguments

  • model::ModelData: The model data container containing the model parameters and arrays.

Keyword Arguments

  • iuse_harmonic_avg_normal_viscosity::Int64:
    • Harmonic averaging of shear viscosity for normal viscosity: 0 off; 1 on
  • iuse_initial_temp_interp::Int64:
    • Interpolate nodal temperatures back to markers at start: 0 off; 1 on
